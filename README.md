# 🗄️ Banco de Dados — Estudos com SQLite

Projeto de estudos para praticar operações com banco de dados
usando Python e SQLite puro, sem ORM.

## 📚 O que foi praticado

- Criação de banco de dados e tabelas (`create_db.py`)
- Inserção de registros (`write_db.py`)
- Consulta de dados (`read_db.py`)
- Atualização de registros (`update_db.py`)

## 🛠️ Tecnologias

- Python 3
- SQLite3 (biblioteca nativa do Python)

## ▶️ Como executar

```bash
git clone https://github.com/caioz-dev/banco_de_dados.git
cd banco_de_dados
python create_db.py  # cria o banco
python write_db.py   # insere dados
python read_db.py    # lê os dados
python update_db.py  # atualiza dados
```
